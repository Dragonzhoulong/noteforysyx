### 表达式解析
>主操作符的概念
```
1+3*4
=>
	+
1       3*4
```
即+作为著操作符, 等左右两递归的返回值进行运算
1. 左结合性 
	>同一级别, 谁后谁作为主操作符
	
```
1-10+3
=>
		+
1-10       3
---------------------------
1-10-10
=>
		-
1-10          10
```
1. 操作符优先级
```
      prior1:+,-

      prior2:*,/

      prior3:!,~,()
```
